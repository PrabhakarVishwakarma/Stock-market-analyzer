Stock Market Analyzer with Machine Learning
An end-to-end Data Science and Machine Learning project that analyzes stocks using Fundamental, Technical, and Risk Analysis, and predicts the next-day stock price using an ML model.

Features
1. Business & Fundamental Analysis
* Company overview (sector, industry, market cap)
* Key financial ratios:
    * P/E Ratio
    * ROE
    * Debt-to-Equity
    * Profit Margins
* Revenue & Profit trends
2. Technical Analysis
* Moving Averages (SMA 50, SMA 200, EMA 20)
* RSI (Relative Strength Index)
* MACD Indicator
* Bollinger Bands
* Trend Detection (Bullish / Bearish / Sideways)
* Support & Resistance Levels
3. Risk Analysis
* Volatility (Daily & Annual)
* Beta vs Benchmark
* Sharpe Ratio
* Maximum Drawdown
* Value at Risk (VaR)
4. Machine Learning Prediction
* Model: Random Forest Regressor
* Predicts next-day closing price
* Feature Engineering:
    * Technical Indicators
    * Returns & Volatility
* Evaluation Metrics:
    * MAE (Mean Absolute Error)
    * RMSE (Root Mean Squared Error)
    * R² Score
5. Final Decision System
* BUY / SELL / HOLD recommendation
* Based on:
    * ML Prediction
    * Technical Trend
    * Risk Profile
6. Visualization
* Price + Moving Averages
* RSI & MACD charts
* Volume analysis
* Drawdown graph
* Returns distribution

Tech Stack
* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* yFinance
* Scikit-learn
